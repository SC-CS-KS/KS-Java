# 线程工具
```md
线程工具其实就是对锁的不断变种，适应更多的开发场景，提高性能，提供更方便的工具，
从最粗暴的同步修饰符，到灵活的可重入锁，到宽松的条件，
接着到允许多个线程访问的信号量，最后到读写分离锁。
```
