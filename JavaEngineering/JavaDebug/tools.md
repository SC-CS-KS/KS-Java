# Java Debug Tools

* btrace
```md
动态地跟踪java运行时程序，将定制化的跟踪字节码切面注入到运行类中，
对运行代码无侵入，根据官方信息判断对性能上的影响也可以忽略不计。
```
* jstatd
```md
与jstat相同，但是可以在线下用客户端连接，可线下操作。
```
* jad
```md
反编译工具，可以将字节码的二进制类反编译回Java源代码。
有些时候项目依赖了没有源代码的jar包，如果出现问题需要定位，那么jad命令可是个小倚天剑。
```
