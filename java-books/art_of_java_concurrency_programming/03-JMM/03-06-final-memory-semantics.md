## final 域的重排序规则
* 对 final 域，编译器和处理器 遵循连个重排序规则
```md
1. 在构造函数中初始化 final（写入），与随后吧这个被构造的对象的引用赋值给一个引用变量，
  这两个操作之间不能重排序。
2. 初次读一个包含 final 域的对象的引用，与随后初次读这个 final 域，不能重排序。
```
